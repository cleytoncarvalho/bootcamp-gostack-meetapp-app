# Meetapp

Final React Native project developed at the GoStack bootcamp. Mobile version of the application for meetup appointments.

Techs:

- React Native
- React Hooks
- React Navigation
- Redux
- Redux Saga
- Redux Persist
- Reactotron
- Styled Components
- Toastify
- Yup
- Axios
- Prop Types
- date-fns
- ESLint
